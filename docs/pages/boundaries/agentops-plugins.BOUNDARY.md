# Orloj Plugins Boundary

Intended location: root of private `orloj-plugins` repository.

## Allowed Scope

- Premium connectors and integration adapters.
- Signed plugin packaging and distribution metadata.
- Versioned compatibility layers for commercial integrations.

## Integration Rules

- Plugins must target published OSS extension/provider contracts.
- Avoid private fork coupling to core runtime internals.
- Keep plugin compatibility matrix aligned with OSS release versions.
