# Resource examples

Each subdirectory holds **one `kind` of Orloj resource** (YAML you can apply with `orlojctl apply -f`). Names and references match the rest of the repo docs.

For composed **topologies**, see [`../blueprints/`](../blueprints/README.md). For **end-to-end scenarios** with prefixed resource names, see [`../use-cases/`](../use-cases/README.md).
